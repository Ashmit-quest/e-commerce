# LUMÉN Store Immersive App State (context.md)

LUMÉN is a premium full-stack metrics platform with an ambient, design-first candlelight theme.

## Features Completed
1. **Interactive Navigation & Views**: Fully integrated router controlling standard dashboard view, detailed orders browser, interactive scent product inventory, customers profiles, funnel analytics, and scheduled marketing campaigns.
2. **MongoDB Database Seeding & Models**: Real-time hot-reloaded REST API endpoints for tracking e-commerce metrics directly inside MongoDB database.
3. **Reactive Actions**:
   - Create new campaigns through the launch modal.
   - Insert new customers dynamically.
   - Live modify product inventory, adjusting price, stocks, and state.
   - Real-time simulated orders arriving with glowing toast updates and automatic notifications synchronization.
   - Live notification banner counts, marked read dynamically.
4. **Visuals & Motion**: Smooth CSS-based flickering candle flame, floating colorful ambient background blobs, overlayed SVG noise grains, custom scrollbars, and fluid spring animations.
